[b]Ferrum Sanctum Tamer[/b]

Tiny Unity Mod Manager mod for Warhammer 40,000: Rogue Trader that tames the Ferrum Sanctum overhead icon and, optionally, its mechanical shock-damage penalty.

[b]Features[/b]
[list]
[*]Hides only the Ferrum Sanctum buff/debuff icon above characters' heads.
[*]Does not suppress other overhead buffs or debuffs.
[*]Optional mechanical modes:
[list]
[*]Vanilla: use the game's normal value
[*]Cap at 10% per stack
[*]Cap at 5% per stack
[*]Disabled: 0% per stack
[/list]
[*]The 10% and 5% modes are caps, not forced values. If the game or a talent lowers Ferrum Sanctum below the selected cap, the lower game value is preserved.
[*]In 0% mode, blocks new Ferrum Sanctum applications and strips existing Ferrum Sanctum buffs from loaded characters.
[*]Restores Ferrum Sanctum buffs removed or blocked by the mod when switching back from 0%.
[*]Updates the Ferrum Sanctum tooltip so it shows the current selected percentage.
[/list]

[b]Install[/b]

Rogue Trader includes the Unity Mod Manager loader used by this mod. You do not need to download the standalone Unity Mod Manager installer.

[list=1]
[*]Launch Rogue Trader at least once, then close the game.
[*]Download [b]FerrumSanctumTamer-1.0.1.zip[/b].
[*]Open this folder, creating it if needed:
[code]%USERPROFILE%\AppData\LocalLow\Owlcat Games\Warhammer 40000 Rogue Trader\UnityModManager\FerrumSanctumTamer[/code]
[*]Extract the contents of [b]FerrumSanctumTamer-1.0.1.zip[/b] directly into that [b]FerrumSanctumTamer[/b] folder.
[*]After extraction, the folder should contain these files directly:
[code]FerrumSanctumTamer.dll
Info.json
README.md
LICENSE[/code]
[*]Make sure you do [b]not[/b] have an extra nested folder like:
[code]FerrumSanctumTamer\FerrumSanctumTamer\Info.json[/code]
[*]Start the game.
[*]Open the Unity Mod Manager overlay in-game with [b]Ctrl+F10[/b].
[*]Configure Ferrum Sanctum Tamer from the mod settings panel.
[/list]

When updating, close the game first, replace the files in the same folder, and then start the game again.

[b]Compatibility[/b]
[list]
[*]Uses Rogue Trader's bundled Unity Mod Manager loader.
[*]Built for Warhammer 40,000: Rogue Trader.
[*]Only targets the Ferrum Sanctum blueprint GUID [code]780497b04a0944e59cb57e68bb9775c4[/code].
[/list]
